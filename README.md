# Omama Bakery

Website omama bakery

## Installation

-   Duplikat file .env.example dan rename menjadi .env dan atur isi file .env sesuai konfigurasi yang dibutuhkan

```bash
  cp .env.example .env
```

-   Generate Key

```bash
  php artisan generate:key
```

-   Migrate database

```bash
  php artisan migrate
```

-   Link Storage

```bash
  php artisan storage:link
```
